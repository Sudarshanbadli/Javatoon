Quick Syntax

public class Main { public static void main(String[] args) { System.out.println("Hello Java"); } }

Main Method: public static void main(String[] args)

Print: System.out.println("Hello");

Variable: int age = 25;

Constant: final double PI = 3.14;

OOP Concepts

Class: class Dog { }

Object: Dog myDog = new Dog();

Inheritance: class Labrador extends Dog

Interface: class Dog implements Pet

Constructor: public Dog() { }

Tip: Use @Override annotation when overriding methods.

Key Terms

  • Bytecode: The platform-independent code (.class) that runs on JVM
  • JVM: Java Virtual Machine - executes bytecode
  • JRE: Java Runtime Environment - libraries + JVM
  • JDK: Java Development Kit - compiler + JRE
  • GC: Garbage Collector - automatic memory management
  • Exception: Runtime error that can be handled

Common Exceptions

NullPointerException - Accessing null object

ArrayIndexOutOfBoundsException - Invalid array index

ArithmeticException - Division by zero

ClassCastException - Invalid type casting

IOException - Input/Output errors

try { // risky code } catch (Exception e) { // handle error } finally { // always executes }

Collections Framework

List: Ordered, allows duplicates (ArrayList, LinkedList)

Set: No duplicates (HashSet, TreeSet)

Map: Key-Value pairs (HashMap, TreeMap)

// ArrayList example ArrayList<String> list = new ArrayList<>(); list.add("Java"); list.add("Python");

Threading Basics

Thread States: New, Runnable, Running, Blocked, Terminated

Create Thread: Extend Thread class or implement Runnable

// Using Runnable Thread t = new Thread(() -> { System.out.println("Thread running"); }); t.start();
Tip: Prefer implementing Runnable over extending Thread for better design.
